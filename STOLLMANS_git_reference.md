
##Notes
- git commit records a snapshot of all tracked files in directory
- Commits get compressed into a set of changes to remain lightweight
- git branches can be changed with independently with individual commits
- I might work on a new "feature branch" in order to add a feature to an app
- Branches include all commits of parent branch plus the new commits
- Command: git checkout -b [branchName] to create a new branch and switch to it
- git push to push changes to remote
- git pull to pull changes from remote
